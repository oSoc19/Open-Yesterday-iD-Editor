Open Heritage Map

Preserve Cultural Heritage

Many memories of industrial buildings that don't exist anymore get forgotten. Open Heritage Map is an online editor where users can preserve these memories.

See it in action Contribute

The Project

Open Heritage Map is a proof of concept for an online map editor that allows users to pinpoint industrial heritage on a map and add information to it, like pictures and stories. This project has been developed during open Summer of code 2019 in collaboration with erfgoed zuidwest.

How It Works

The editor is based on iD (v2.15), a friendly JavaScript editor for OpenStreetMap; the information shared using our tool is uploaded to the Open Historical Map servers to ensure it can live on in the cloud, forever.
